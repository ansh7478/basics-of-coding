# Indian Recipe Book

## Project Description

The Indian Recipe Book is a basic functional programme developed using Python and Streamlit. 
It allows users to view existing Indian recipes and add their new recipes to the collection of recipes.

The programme stores recipe information in a JSON file, which allows the data to remain available after the application is even closed.

## Main Features of this project

- Can view all available recipes
- Select and display a specific recipe
- View recipe ingredients
- View cooking instructions
- Add a new recipe
- Save recipe data permanently in a JSON file
- Simple and user-friendly interface

## Technologies that are Used

- Python
- Streamlit
- JSON
- Visual Studio Code
- GitHub


## How to Run

Install Streamlit:

pip install -r requirements.txt

Run the project:

streamlit run app.py

## Files

- app.py – Main program
- recipes.json – Stores recipe data
- requirements.txt – Required library
